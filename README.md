# Overview

To further my learning this sprint, I decided to make a webapp.

This webapp is supposed to be a place to store music and make playlists.

The reason I wanted to write this webapp was so I could learn what writing a webapp entails.

# Web Pages

When first trying to access this webapp, it will ask you to login or create an account. Since this is just a demonstration that won't be hosted on a live server, I set the admin username to admin and the admin password is 12345 so people can copy this code if they want. After getting logged in, you get to the home page for the user, which has a link to a page to view all of the playlists. When clicking on that, you are sent to a list that allows you to add or remove playlists. You can then click one of the playlists to add songs to it.
